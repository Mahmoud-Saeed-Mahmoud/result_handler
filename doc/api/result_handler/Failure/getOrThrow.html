Returns the value held by a Success, or throws the error wrapped in an Exception if the Result is a Failure.

Use this when you expect a success and consider a failure an exceptional case. Returns the success value. Throws if this is a Failure.

Implementation

@override
T getOrThrow() {
  // Wrap the error in Exception before throwing it,
  // since the error itself might not be an object that can be directly thrown
  // or to provide a common exception type.
  throw Exception(error
      .toString()); // Or just `throw error;` if E is always an Exception type.
}
