Binds a function to the Result. If the result is a Success, the function is applied to the value, returning a new Result. If the result is a Failure, the Failure is returned without modification.

This is similar to flatMap, but is more aligned to functional programming and allows you to compose the code and the chain of operations in a more descriptive way.

transform is the function to bind to the Success value. Returns a new Result based on the transformation.
