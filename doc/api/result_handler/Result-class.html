Result

Result<T, E> class abstract

An abstract class representing the result of an operation, which can either be a Success or a Failure.

T is the type of the value held by a Success. E is the type of the error held by a Failure.

Binds a function to the Result. If the result is a Success, the function is applied to the value, returning a new Result. If the result is a Failure, the Failure is returned without modification.
