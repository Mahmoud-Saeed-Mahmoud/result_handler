bind<R>(Result<E, R> transform(T value)) → Result<E, R>
Binds the given function to the value of this result, if it is a success. If this result is a failure, the given function is not called and a new failure with the same error is returned. If this result is a success, the given function is called with the value of this result and the result of that call is returned as the result of this function.
