Factory method to create a Result by executing a synchronous function fn.

If fn completes successfully, its result is wrapped in a Success. If fn throws an exception, onError is called with the caught error and stack trace to produce an error value of type E, which is then wrapped in a Failure.

fn The synchronous function to execute. onError A function that converts a caught Object and StackTrace into an error of type E. Returns a Result<E, T>.

Implementation

static Result<E, T> tryCatch<E, T>(
  T Function() fn,
  E Function(Object error, StackTrace stackTrace) onError,
) {
  try {
    return Success(fn());
  } catch (e, s) {
    return Failure(onError(e, s));
  }
}
